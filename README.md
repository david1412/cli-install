#  Installs Azure CLI, Helm, kubectl, Terraform, and Crossplane on Linux / Ubuntu / RHEL / SUSE

Clone the repo and run the bash script to install CLI's.

```
git clone https://github.com/david1412/cli-install.git
cd install
chmod +x script.sh
./script.sh
```

# Check installation 

> terraform version

> helm version

> kubectl version

> az --version
